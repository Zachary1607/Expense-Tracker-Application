# Expense Tracker Application

A full-stack expense management application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application helps users manage income and expenses, categorize transactions, and view financial summaries through a simple dashboard.

## Features

- User authentication and account management
- Add, edit, and delete expense records
- Categorize transactions
- View income and expense summaries
- Track spending patterns
- Responsive user interface
- REST API integration
- MongoDB data storage

## Tech Stack

### Frontend
- React.js
- Redux
- React Bootstrap
- Material Icons

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
